# Project Overview

Running Signals is a live analytics engineering portfolio project.

The project turns personal Garmin running data into analytical signals for consistency, volume, and fitness.

It is intentionally scoped as an analytics engineering project rather than a running product.

## Signal Groups

### Consistency

Shows whether training habits are stable.

### Volume

Shows how much work is being accumulated over time.

### Fitness

Shows directional aerobic performance using pace/heart-rate efficiency and Garmin Recovery HR when it
is available in FIT event payloads. Same-day Garmin Connect health context, including resting heart
rate, HRV, and sleep fields, is modeled as descriptive context rather than readiness scoring.
